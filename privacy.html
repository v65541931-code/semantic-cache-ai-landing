Privacy Policy

Last Updated: May 2026

Semantic Cache AI ("we", "our", or "the extension") is built with a local-first architecture. We take your privacy extremely seriously.

1. Data Collection & Storage

All data processed by Semantic Cache AI, including your chat prompts, AI responses, API keys, and Master Passwords, is stored 100% locally on your device using browser storage and the Origin Private File System (OPFS).

We do not operate a central database. We do not transmit your chat history or API keys to any external servers belonging to us.

2. API Keys & Authentication

If you use the Pro tier, your third-party API keys (e.g., OpenAI, Google) are encrypted locally using AES-GCM cryptography. The Master Password used to encrypt these keys never leaves your machine.

3. Third-Party Services

If you enable Neural Semantic Caching, the extension makes direct API calls from your local browser to the AI providers (OpenAI or Google) using the API keys you provide. These requests are governed by the respective privacy policies of those providers.

4. Selling of Data

We certify that we do not sell, rent, or transfer your data to third parties. We do not use your data for advertising, lending, or any purposes unrelated to the core functionality of the extension.

5. Contact

If you have any questions regarding this privacy policy, please contact the developer via the Chrome Web Store support tab.
